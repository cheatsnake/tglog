# Telegram Logger

A tiny module for sending logs to Telegram chats.

## Installation

```bash
npm install tglog
```

Import into your project:

```js
// ESM
import TelegramLogger from "tglog";
```

```js
// CommonJS
const { default: TelegramLogger } = require("tglog");
```
